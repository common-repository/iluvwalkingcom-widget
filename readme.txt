Allows you to display your walking statistics from ILuvWalking.com

== Description ==

ILuvWalking.com is the number one site for tracking your steps online.  With their easy to use interface and great 
content, you can't go wrong!  

This plugin gives you a widget that allows you to show your step statistics on your own WordPress blog.  Your updated
statistics will be shown immediately after you enter new data on ILuvWalking.com

This widget requires cURL to be installed on your server or allow_url_fopen to be set on.  If neither of these 
prerequisites are met, then the widget will not display. 

== Installation ==

1. Upload `post-satisfaction` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the widget via the `Widgets` menu by entering a title (if you want) and your ILuvWalking.com username.
